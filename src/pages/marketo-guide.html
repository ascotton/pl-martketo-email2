Modules

Modules must be added in Marketo by duplicating a current module and dragging it to the place in the email you desire. Use the Maketo panels on the right side of your screen to modify variables for these modules. See that name in the upper left of this box? (Demo Module) That's the name of this module.

 Tags

Tags are standard HTML tags like H1, P, Strong, etc that have build in styling. An example would be:
<p> I am a paragraph!</p>

Do not attempt to add any styling to these tags, its already in the template (except for the classes below). And do not use any tags not outlined below. Classes

classes must be added to a preexisting HTML tag from above or by wrapping an element in a DIV tag you use the syntax 'class="name_of_class_here here_is_another_class"' example:
<div class="text-left"> I'm going to go ahead and align left now</div>

Warning! Do not use any other classes than those outlined below.
